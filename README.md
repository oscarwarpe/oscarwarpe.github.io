# oscarwarpe.github.io

Visittkort for Oscar Warpe. Se alle prosjektene på Warpedev.

## Struktur

- `index.html` – statisk rot med seksjoner for intro, prosjekter og kontakt.
- `styles/main.css` – sideoppsett, importerer felles identitet fra `../../assets/styles/identity.css`.
- `scripts/main.js` – navscroll, typingsløyfe og partikkelbakgrunn (via `../../assets/scripts/ambient.js`).

## Lokalt

Åpne `index.html` i nettleser. Endringer i `Dev/assets/` deles med andre Presence-prosjekter;
test lokalt før du pusher slik at relative stier fungerer på GitHub Pages.
